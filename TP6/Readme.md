# Modelos Probabilísticos en PLN - N-gramas

Este repositorio contiene una práctica de **Procesamiento del Lenguaje Natural (PLN)** con **modelos probabilísticos de N-gramas**.  

## Contenido
- `CorpusEducacion.txt`: corpus de opiniones de estudiantes.
- `Ngramas.ipynb`: notebook con el código para generar y comparar 2-gramas y 3-gramas.
- `README.md`: instrucciones de uso.

## Ejecución
1. Abrir Jupyter Notebook.
2. Cargar `Ngramas.ipynb`.
3. Ejecutar todas las celdas.
4. Se generarán gráficos comparativos de 2-gramas y 3-gramas con frecuencia mínima (`min_df=2`).

## Resultados
- Limpieza del corpus con tokenización, lematización y eliminación de stopwords.
- Conteo de frecuencias absolutas y relativas.
- Gráficos de barras comparando 2-gramas y 3-gramas.
