# Modelo de Espacio Vectorial

Este proyecto implementa un sistema de **búsqueda de similitudes entre documentos** utilizando el **Modelo de Espacio Vectorial** con la librería **scikit-learn** en Python.

---

## 📂 Contenido
- `EspacioVectorial_Consigna3.ipynb`: notebook con la implementación paso a paso.
- `README.md`: este archivo con instrucciones y ejemplos.

---

## 🚀 Ejecución
1. Abrir Jupyter Notebook.
2. Cargar `EspacioVectorial_Consigna3.ipynb`.
3. Ejecutar todas las celdas.
4. Se mostrarán:
   - Los vectores TF-IDF de cada documento.
   - La matriz de similitud del coseno.
   - Un gráfico de la matriz de similitud.
