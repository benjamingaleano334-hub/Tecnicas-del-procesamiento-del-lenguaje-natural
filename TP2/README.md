# Gramáticas Libres de Contexto — Resolución de Ejercicios

Este proyecto fue realizado por **Benjamín Galeano** en el marco de la materia **Técnicas del Procesamiento del Lenguaje Natural**.  
El objetivo central fue **responder las preguntas teóricas y prácticas** relacionadas con una gramática libre de contexto (GLC), mostrando paso a paso cómo se derivan cadenas y cómo se describe el lenguaje generado.

## 📘 Lo que se hizo
- Se definió formalmente la **gramática G**, con sus variables, terminales y símbolo inicial.  
- Se respondió a cada una de las preguntas planteadas:
  - Identificación de variables y terminales.  
  - Determinación del símbolo inicial.  
  - Ejemplos de cadenas válidas en el lenguaje.  
  - Obtención de la cadena mínima posible.  
  - Evaluación de derivaciones y producciones como verdaderas o falsas.  
  - Construcción de un **árbol de derivación** para una cadena específica.  
  - Descripción en español del lenguaje L(G).  
- Se organizó todo en un documento académico con formato de infografía.

## 📂 Archivos
- `gramaticas_libres_de_contexto_benjamin_galeano.pdf` → Documento principal con todas las respuestas y el análisis completo.  

## 🎯 Propósito
Este trabajo muestra la capacidad de aplicar conceptos de **gramáticas libres de contexto** en un entorno académico, resolviendo ejercicios paso a paso y explicando el razonamiento detrás de cada respuesta.
